# WatermarkText
Add text to photo

### Usage

#### 文字水印

* `python watermark.py`
* input your image url
* input text you want mark
* finish! See the output file.

#### 图片水印
* `python watermark_image.py`
* input your image url
* input your watermark image url
* choose the position
* finish! See the output file.

如果报错应该是python版本不对，需要用python3
